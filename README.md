# API-Hunter
API Hunter is a client-side application that interfaces with the official GitHub REST API. This module fetches remote data via HTTP requests and render it to the DOM asynchronously.

## Tech Stack
HTML5, CSS3 (Flexbox + Grid), Vanilla JavaScript, GitHub REST API

## Features
- Dark/Light mode toggle
- Lighthouse 100/100 Performance & Accessibility
- Search GitHub users by username and display their profile card with avatar, name, bio, join date, and portfolio link
- Fetch and display top 5 latest repositories with links
- Battle Mode — compare two users' total repository stars
- Responsive design across mobile and desktop
